
# Gather Note

作成したノートを**グループ内で共有できる**アプリです。

[https://gathernote.vercel.app/](https://gathernote.vercel.app/)

## 使用技術

1. Next.js
2. TypeScript
3. supabase
4. swr
5. zustand
6. OpenAi Api
